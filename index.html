About Me

UI-focused engineering with a performance and accessibility mindset.

I’m a Frontend Software Engineer who enjoys translating product goals into clean, reliable interfaces. I have a strong foundation in HTML, CSS, and JavaScript (ES6+), and I build components with a deep respect for semantics, responsive layout systems, and readable code.

I’m experienced with React and modern UI patterns—component composition, state management, and reusable design systems. I prioritize accessibility (semantic HTML, ARIA when needed, keyboard navigation) so experiences work well for every user.

I’m performance-aware and detail-oriented: optimizing images, reducing layout shift, trimming unnecessary JavaScript, and measuring results in Chrome DevTools. I work comfortably with designers and backend engineers to ship polished, scalable features.
